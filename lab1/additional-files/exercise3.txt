Exercise 3
----------

Q1: Use the Bochs debugger to trace into the ROM BIOS for a few more
instructions, and try to guess what it might be doing.

(0) [0xfffffff0] f000:fff0 (unk. ctxt): jmp far f000:e05b         ; ea5be000f0
  Quando a máquina é resetada ou ligada, o PC é setado para o final da área
reservada para o código da BIOS. Então a primeira coisa que a BIOS faz é
'pular para trás', assim ela fica com mais espaço para seus dados e código.

(0) [0x000fe05b] f000:e05b (unk. ctxt): xor ax, ax                ; 31c0
  Uma operação XOR com o mesmo número resulta em 0, então esse código coloca
0 em AX. Essa instrução é usada porque seu código de máquina é menor
do que a versão correspondente usando MOV.

(0) [0x000fe05d] f000:e05d (unk. ctxt): out 0x0d, al              ; e60d
  Faz um Master Clear na primeira controladora de DMA. Tem o mesmo efeito de
um reset de hardware, vários registradores são resetados e o chip entra em
'idle cycle'.
  Note que a operação é ativada pela própria operação de I/O, o valor no
registrador não importa.

(0) [0x000fe05f] f000:e05f (unk. ctxt): out 0xda, al              ; e6da
  Master Clear na segunda controladora de DMA.

(0) [0x000fe061] f000:e061 (unk. ctxt): mov al, 0xc0              ; b0c0
(0) [0x000fe063] f000:e063 (unk. ctxt): out 0xd6, al              ; e6d6
  Escreve 11000000 no mode register da segunda controladora de DMA, essa
operação tem os seguintes efeitos: cascade mode, address increment select,
autoinitialisation disable, verify operation, channel 4 select.

(0) [0x000fe065] f000:e065 (unk. ctxt): mov al, 0x00              ; b000
(0) [0x000fe067] f000:e067 (unk. ctxt): out 0xd4, al              ; e6d4
  Escreve 00000000 no write single mask register, esse operação tem os
seguintes efeitos: seleciona canal 4 e limpa o master bit.


(0) [0x000fe069] f000:e069 (unk. ctxt): mov al, 0x0f              ; b00f
(0) [0x000fe06b] f000:e06b (unk. ctxt): out 0x70, al              ; e670
    Escreve 00001111 no CMOS RAM index register port, essa operação tem os
seguintes efeitos: habilita NMI e escreve 0001111 no index.

(0) [0x000fe06d] f000:e06d (unk. ctxt): in al, 0x71               ; e471
(0) [0x000fe06f] f000:e06f (unk. ctxt): mov bl, al                ; 88c3
    Lê o segundo atual (em BCD) em AL, e copia o resultado para BL.
